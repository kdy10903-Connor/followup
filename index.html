기획서 — 개요 · 목적 · 구성
MPF 추적관리 대시보드
사업·마케팅·재무·Vision 2030을 하나의 흐름으로 추적하는 전사 합의 지표 체계
목적

경영의 성공은 추적의 성공이다. 흐름이 꺾이기 전에 알아채야 다음 달을 바꿀 수 있다.

이 대시보드는 사업·마케팅·재무의 핵심 지표를 하나의 프레임으로 연결하여, 경영위원회가 이상신호를 즉각 감지하고 경영전략실이 원인을 정확히 진단할 수 있도록 설계되었다. 국가별 필터로 전체 판과 개별 국가를 자유롭게 전환하며 볼 수 있다.

목표
목표 01
이상신호 조기 감지
흐름이 꺾이기 전에 월간 리뷰에서 포착. 경영위원회 전용 Layer 1 지표.
목표 02
원인 정확 진단
이상신호 발생 시 경전실이 Layer 2로 한 단계 더 깊이 파고드는 구조.
목표 03
전사 지표 통일
부서마다 다른 기준 제거. 전사 합의 지표로 추적 일관성과 누적 학습 확보.
목표 04
Vision 2030 연결
매월의 성과가 2030 목표 궤도 위에 있는지 CAGR 선으로 즉시 확인.
전체 지표 목록
페이지 지표명 무엇을 보는가 국가별 분리
기획서 — 지표 연결 구조
어디가 꺾이면 무엇이 영향받는가
두 개의 메인 흐름이 합쳐져 전체 매출·영업이익률로 수렴한다
신규 흐름
MBM 참여율 → DB 확보 / ICP Rate → aIDMA 전환율·체류시간 → Sales Cycle Length → 신규 매출
MBM 참여율이 떨어지면 → DB 확보 감소 → ICP Rate 하락 시 퍼널 오염 → aIDMA D→M 단계 정체 → Sales Cycle 연장 → 신규 매출 감소로 이어진다. 선행 지표가 후행 결과보다 2~3개월 앞서 신호를 보낸다.
기존 흐름
DAU/MAU · 사용량 → 재계약 aIDMA 전환율 → MODS 유지율 · ARPU → 기존 매출
DAU/MAU와 사용량이 감소하면 → 재계약 aIDMA 의향 단계에서 이탈 증가 → MODS 유지율 하락 → 기존 매출 감소로 이어진다. 사용량 감소는 이탈의 1~2분기 선행 신호다. 가격 문제가 아닌 사용 경험 문제일 가능성이 높다.
수렴 → 재무
신규 매출 + 기존 매출 → 전체 매출 · 비중 → 영업이익률 · 인당 영업이익
보완 필요: Sales Cycle 지연 → 매출 인식 지연 → 현금 유입 지연 → 자금 현황으로 이어지는 연결이 현재 대시보드에서 자동으로 보이지 않는다. 단기: P0 신호등에서 Sales Cycle 임계값 초과 시 자금 현황 지표에도 주의 표시 연동 예정.
기획서 — 용어 정의
지표 약어 풀네임 및 정의
대시보드에 등장하는 모든 약어의 풀네임과 의미
MPFMonitoring · Prediction · Feedback
경영 추적의 3단계 프레임. 모니터링으로 이상 감지, 예측으로 선행 판단, 피드백으로 개선 반영.
aIDMAawareness · Interest · Desire · Motivation · Action
마이다스형 영업·마케팅 퍼널. 인지→관심→욕구→동기→행동(계약) 5단계 전환 추적.
MODSMaintenance On Demand Service
연간 유지보수 계약. 기존 고객 매출의 핵심 구조. MODS 유지율 = 갱신 비율.
ARPUAverage Revenue Per User
사용자당 평균 매출. 제품별·국가별로 분리하여 운영 효율성 파악.
CACCustomer Acquisition Cost
신규 고객 1건 획득 비용. 마케팅+영업 총비용 ÷ 신규 계약 건수.
ICP RateIdeal Customer Profile Rate
확보된 DB 중 이상적 고객 프로파일 부합 비율. 양이 아닌 질 지표.
TAMTotal Addressable Market
전체 확보 가능 시장. RPM 기준 전세계 엔지니어 약 100만명.
MBMMaximize by Marketing
마케팅으로 극대화. 세미나·웨비나 등 마케팅 활동을 통해 사업 성과를 극대화하는 전략. 참여율과 후속 DB 전환율로 효과 측정.
DAU/MAUDaily / Monthly Active Users
일간/월간 활성 사용자 수. 비율이 높을수록 제품이 일상에 정착된 상태.
SCLSales Cycle Length
첫 접촉(a)부터 계약(A)까지 평균 소요 일수. 단축이 핵심 과제.
CAGRCompound Annual Growth Rate
연평균 복합 성장률. 2030 목표 역산 시 필요 CAGR vs 현재 실제 CAGR 비교로 궤도 이탈 판단.
YoYYear over Year
전년 동기 대비 성장률. 계절성을 제거한 성장 추이 비교 기준.
